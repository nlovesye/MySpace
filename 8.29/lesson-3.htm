基本标签（结构标签）

  • 常用标签 <html>、 <head>、<body>
  • 注意事项：
    • 都是双标签
    • 只能出现1次

标题标签

  • 常用标签 <title>
  • 注意事项：
    • 双标签
    • 只能出现1次

文本标签

  • 常用标签 <pre> 、<h*> 、<b> 、<i> 、<em>、<strong>、<font>
  • 注意事项：
    • 双标签
    • pre是创建预格式化文本
    • h是块级元素会换行、其余不会，h标签默认有加粗样式
    • 行内元素标签之间如果有空白的话，只显示一个空格
    • b、i对应的是strong、em,后两者对搜索引擎更友好
    • h*(1~6)是块级元素，默认有加粗样式。一个页面只能有一个h1标签
    • 可以出现多次

链接标签

  • 常用标签 <a>
  • 注意事项：
    • 双标签
    • <a name="mark">方式可以定义锚点
    • a可以跳转到站内锚点、发送email、qq聊天等
    • 可以出现多次

格式排版

  • 常用标签 <p>、<div>、<span>、<dl>、<dt>、<dd>、<ul>、<ol>、<li>、<br>
  • 注意事项：
    • div,span是没有语义的标签
    • <br>是单标签，其余都是双标签,p虽然单标签双标签都可以，但大家都拿他作为双标签来用
    • dl的子标签只能是dt、dd,ol、ul的字标签只能是li
    • 如果我们布局的区块是一个整体，并且它既有标题又有内容，很适合用dl+dt+dd布局
    • 行内标记不要写在块标签的外部，应该写在内部
    • 可以出现多次

图形标签

  • 常用标签 <img>、<hr>
  • 注意事项：
    • 单标签
    • img标签中alt属性最好加上
    • 可以出现多次

表格标签

  • 常用标签 <table>、<tbody>、<thead>、<tr>、<th>、<td>
  • 注意事项：
    • 常用的属性
      <table border=value> 设置围绕表格的边框的宽度
 <table cellspacing=value>   设置表格格子之间空间的大小
 <table cellpadding=value>   设置表格格子边框与其内部内容之间空间的大小
 <table width=value 或 %> 设置表格的宽度－用绝对像素值或文档总宽度的百分比
 <tr align=……>   设置表格格子的水平对齐（左中右）（left center right）
 <tr valign=……>  设置表格格子的垂直对齐（上中下）（left center right）
 <td colspan=value>  设置一个表格格子应跨占的列数（缺省为1）
 <td rowspan=value>  设置一个表格格子应跨占的行数（缺省为1）
 <td nowrap> 禁止表格格子内的内容自动断行回卷
    • colspan 合并同一行内的相邻单元格 max:总列数 min:2
    • rolspan 合并同一列的相邻单元格 max:总列数 min:2

表单标签

  • 常用标签 <form><select><input><option><textarea><textarea><button><label>
  • 注意事项：
    • input、button单标签，其余双标签
    • method为发送请求的方式，get表示通过地址栏方式，post表示以 HTTP post 事务的形式发送表单数据（form-data）
    • 常用标签及属性
      <form>……</form> 创建所有表单
 <select multiple name="NAME" size=value>………</select>    创建一个滚动菜单，size设置在需要滚动前可以看到的表单项数目
 <option>    设置每个表单项的内容
 <select name="NAME">……</select> 创建一个下拉菜单
 <textarea name="NAME" cols=value rows=value>……</textarea>   创建一个文本框区域，列的数目设置宽度，行的数目设置高度
 <input type="checkbox" name="NAME"> 创建一个复选框，文字在标签后面
 <input type="radio" name="NAME" value="x">  创建一个单选框，文字在标签后面
 <input type=text name="……" size=value>  创建一个单行文本输入区域，size设置以字符计的宽度
 <input type="submit" value="NAME">  创建一个submit（提交）按钮
 <input type="image" border=0 name="NAME" src="/name.gif">   创建一个使用图象的submit（提交）按钮
 <input type="reset">    创建一个reset（重置）按钮

多媒体标签

  • 常用标签 <object>、<embed>
  • 注意事项：
    • 双标签
    • OBJECT和EMBED标签
    • 可以出现多次

注释标签

  • 常用标签 <!-- -->;
  • 注意事项： 代码注释的作用是帮助程序员标注代码的用途，过一段时间后再看你所编写的代码，就能很快想起这段代码的用途。代码注释不仅方便程序员自己回忆起以前代码的用途，还可以帮助其他程序员很快的读懂你的程序的功能，方便多人合作开发网页代码。

扩展属性标签

  • 常用标签 <base> 、<link> 、<meta> 、<style> 、<script>
  • 注意事项：
    • script、style为双标签，其余都为单标签
    • base href属性（页面中所有相对链接的基准URL）影响的是有href、src属性的标签，target影响的是有target属性的标签
    • link引入外部样式文件，style放置内部样式表，script放置外部和内部Javascript
    • meta 
      <meta name="keywords" content="HTML标签大全（类型）,日志,Claudine,忘忧屋,网易博客,网易,blog"/>
 <meta name="description" content="HTML标签大全（类型）,Claudine的网易博客,执子之手 与子偕老,咱是淳朴的孩子！！"/>
 <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
 <meta http-equiv="content-type" content="text/html;charset=utf-8">
    • 可以出现多次
